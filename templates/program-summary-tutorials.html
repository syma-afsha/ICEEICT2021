Advanced Arduino IoT board hands on tutorial session


Ms Khadija Akhter
Code 19


Biography: Khadija Akhter is an experienced IoT engineer and lead trainer on Arduino boards. She is currently working as an IoT engineer at Code 19 and has extensive experience in IoT based product development for smart agriculture, smart security and automation. Her skills and interest are Arduino, IoT systems, cloud architecture and advanced industry 4.0 technology.


About the session: In this online tutorial session Khadija Akhter will provide an overview of advanced Arduino boards globally used for IoT and industry 4.0. She will guide the audience to configure Arduino MKR series boards for data acquisition and Arduino cloud interface. The audience will get a live demo of an IoT based smart pond management system developed by the Code 19 R&D team.
